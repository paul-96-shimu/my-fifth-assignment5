Question-2: What is the difference between innerHTML, innerText, and textContent ?


Answer: Ther are some different are innerHTML, innerText, and textContent

1. innerHTML: Reads or modifies the HTML inside an element. Parses any HTML tags inside it. javascript Copy Edit

2. innerText: Retrieves only the visible text. Ignores hidden elements (display: none or visibility: hidden). Doesn’t parse HTML.

3. textContent: Retrieves all text content, including hidden elements. Does not parse HTML.

Question-3: What is event delegation in the DOM?


Answer:
Event Delegation is a technique in JavaScript where instead of adding event listeners to multiple child elements, we add a single event listener to a parent element. This works because of event bubbling, where events propagate up from the target element to its ancestors.

Question-4: What is event bubbling in the DOM?


Answer:
Event bubbling is a mechanism in the DOM where an event starts from the target element (where the event occurs) and then propagates up through its parent elements until it reaches the root (document).
